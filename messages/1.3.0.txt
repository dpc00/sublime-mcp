MCP Commander 1.3.0
===================

The MCP server is now built directly into the ST plugin — no external
process (pip or npx) is required.

To connect Claude Code, add to ~/.claude/settings.json:

  Windows:
    "sublime-mcp": { "type": "sse", "url": "http://127.0.0.1:9502/sse" }

  Mac / Linux:
    "sublime-mcp": { "type": "sse", "url": "http://127.0.0.1:9503/sse" }

New: "MCP Commander: Server Status" in the Command Palette lets you
stop or restart the server without reloading the plugin.

Full documentation: https://github.com/dpc00/sublime-mcp
