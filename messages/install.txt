sublime-mcp
===========

Sublime Text MCP server — lets Claude Code (or any MCP client) read and
control a running ST instance via a local HTTP bridge. 63 tools covering
reading, navigation, editing, searching, build, Terminus integration,
settings, layout, and live Python scripting.

The ST plugin is now running. To complete setup:


STEP 1 — Install the MCP server (run once in your terminal)
------------------------------------------------------------

    pip install sublime-mcp


STEP 2 — Register with Claude Code
-----------------------------------

Add to ~/.claude/mcp.json (or ~/.claude/settings.json):

    {
        "mcpServers": {
            "sublime-mcp": {
                "command": "sublime-mcp"
            }
        }
    }

Restart Claude Code. Tools appear with the mcp__sublime-mcp__ prefix.


The plugin listens on 127.0.0.1:9500. Check View > Show Console for:
  sublime-mcp: listening on 127.0.0.1:9500

Full documentation: https://github.com/dpc00/sublime-mcp
