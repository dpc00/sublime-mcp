MCP Commander
=============

Exposes Sublime Text to AI assistants (Claude Code, Cursor, etc.) via the
Model Context Protocol.  The MCP server is built into this plugin — no
separate process to install or manage.

The plugin is now running.  Complete setup in one step:


STEP 1 — Register with Claude Code
------------------------------------

Add to ~/.claude/settings.json under "mcpServers":

  Windows:
    "sublime-mcp": { "type": "sse", "url": "http://127.0.0.1:9502/sse" }

  Mac / Linux:
    "sublime-mcp": { "type": "sse", "url": "http://127.0.0.1:9503/sse" }

Restart Claude Code.  Tools appear with the mcp__sublime-mcp__ prefix.


Server management
-----------------

The MCP server starts automatically when ST loads the plugin.
To stop or restart it, open the Command Palette and run:

  MCP Commander: Server Status


Ports
-----

  MCP SSE (default):   Windows 9502  /  Mac/Linux 9503
  HTTP bridge:         Windows 9500  /  Mac/Linux 9501

Override with env vars SUBLIME_MCP_MCP_PORT or SUBLIME_MCP_PORT.
Check View > Show Console for startup confirmation.


Full documentation: https://github.com/dpc00/sublime-mcp
